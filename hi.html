To style your existing table using a CSS class instead of a style attribute, create a class (let's call it .result-table) and apply it to your <table> tag. Here’s how you can edit your lines:

1. CSS (internal or external style block)

xml
<style>
.result-table {
    border: 2px solid black;
    width: 100%;
    border-collapse: separate;
    border-spacing: 0 16px; /* space between rows */
}
.result-table th {
    border-bottom: 2px solid black;
    padding: 12px 8px;
    background: #fff;
    font-weight: bold;
    text-align: center;
}
.result-table td {
    padding: 10px 8px;
    border: none;
    background: #fff;
    text-align: center;
}
.result-table tr {
    /* No border */
}
</style>

2. HTML

Change your <table style="..."> to <table class="result-table">.

xml
<table class="result-table">
    <tr>
        <th>SEM <br>NO.</th>
        <th>COURSE <br>CODE</th>
        <th>COURSE TITLE</th>
        <th>CREDITS</th>
        <th>LETTER <br>GRADE</th>
        <th>GRADE <br>POINT</th>
        <th>RESULT</th>
    </tr>
    <!-- rest of your table rows -->
</table>

Edit Summary:

  • Remove the inline style attribute from <table>.

  • Add class="result-table" to your <table> tag.

  • Use the .result-table CSS class above for styling.

To add column borders for both the heading and all rows (so each cell has a vertical border), use the following internal CSS and HTML structure:

xml
<style>
.result-table {
    border: 2px solid black;
    width: 100%;
    border-collapse: separate;
    border-spacing: 0 16px;
}
.result-table th, .result-table td {
    border-right: 2px solid black; /* vertical column border */
    padding: 12px 8px;
    text-align: center;
    background: #fff;
}
.result-table th {
    border-bottom: 2px solid black; /* heading horizontal border */
    font-weight: bold;
}
.result-table tr {
    /* No bottom border for rows */
}
.result-table th:last-child,
.result-table td:last-child {
    border-right: none; /* Remove right border for last column */
}
</style>

<table class="result-table">
    <tr>
        <th>SEM <br>NO.</th>
        <th>COURSE <br>CODE</th>
        <th>COURSE TITLE</th>
        <th>CREDITS</th>
        <th>LETTER <br>GRADE</th>
        <th>GRADE <br>POINT</th>
        <th>RESULT</th>
    </tr>
    <!-- Table rows as in your data -->
</table>

  • Every cell (header and row) will have a right-side border, forming column separators.

  • The bottom border for the header distinguishes it from the rest.

  • Space between rows is provided for clarity.

  • The last cell in each row does not show a double right border, so appearance is clean.

Use your normal table rows inside this structure. The class name is .result-table for consistency.
